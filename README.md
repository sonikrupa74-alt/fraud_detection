# Fraud Detection System 🔍

## Overview

This project is a machine learning-based system designed to detect fraudulent financial transactions using classification techniques. It enables real-time fraud prediction through a deployed FastAPI backend.

## Tech Stack

* Python
* pandas, NumPy, scikit-learn
* FastAPI
* Render (Deployment)

## Features

* Real-time fraud detection using API
* Data preprocessing and feature engineering
* Classification model for fraud prediction
* Deployed and accessible online

## Live Demo

https://fraud-detection-uphl.onrender.com

## How It Works

1. Transaction data is sent to the API
2. Data is preprocessed and analyzed
3. Machine learning model predicts fraud or non-fraud
4. Result is returned instantly

## Result

* Built a reliable fraud detection system using machine learning
* Demonstrates end-to-end ML workflow (data → preprocessing → model → API → deployment)
* Handles real-world financial dataset for prediction
